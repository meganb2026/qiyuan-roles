# 奇缘48

一个基于 Twine 框架开发的交互式文字冒险游戏。

## 游戏介绍

这是一个单机文字冒险游戏，背景设定在一个充满阴谋与秘密的皇宫中。玩家将扮演六个角色之一，在三天内完成各自的目标。

## 游戏角色

游戏中有六个可选择的角色，每个角色都有独特的身份、初始装备和目标：

1. **克劳狄斯** - 国王：威严的国王，掌握着王国的最高权力
2. **程婴** - 御医：医术高超但风评阴狠的御医，传说就是他陷害死了公孙杵臼
3. **何非** - 潜水教练：潜水教练，消息非常灵通，什么人都认识。写作江湖百晓生读作江湖骗子
4. **李想** - 市政排水总工程师：发现并揭露排水系统中的秘密
5. **吴智哲** - 建筑设计师：找到师傅留下的秘密并保护它
6. **王卫国** - 施工运输队队长：阻止非法物资的运输并揭露真相

## 玩法说明

- **角色选择**：游戏开始时，从六个角色中选择一个开始游戏
- **三天时限**：游戏共进行三天，每天是一个章节
- **装备交换**：每天白天可以与其他角色交换一次装备，通过策略性的交换来获得完成目标所需的物品
- **目标达成**：在三天内收集到完成角色目标所需的装备即为成功，否则失败
- **策略性**：需要仔细规划每天的交换策略，因为每天只能交换一次

## 运行方式

### 方式一：在线游玩（推荐）

访问上面的游戏链接即可在浏览器中直接游玩，无需下载。

### 方式二：本地运行

1. **准备游戏文件**：
   ```bash
   ./tools/build.sh  # 生成 game.twee 文件
   ```

2. **导出HTML文件**：
   - 打开 [Twine在线编辑器](https://twinery.org/)
   - 点击 **Import From File** 导入 `game.twee`
   - 点击 **Build** → **Publish to File**
   - 保存为 `game.html`

3. **运行游戏**：
   在浏览器中打开 `game.html` 文件

### 方式三：开发模式

1. 确保已安装 Twine 2.x 或使用在线版本
2. 运行构建脚本生成游戏文件：
   ```bash
   chmod +x tools/build.sh
   ./tools/build.sh
   ```
3. 在浏览器中打开生成的 `game.html` 文件即可开始游戏

### 方式二：使用 Twine 编辑器

1. 下载并安装 [Twine 2](https://twinery.org/)
2. 在 Twine 中导入 `src/` 目录下的所有 `.twee` 文件
3. 在编辑器中测试和运行
