# 奇缘6

一个基于音乐剧演员郑棋元老师演出过往角色的单机版小游戏。

## 游戏介绍

这是一个单机文字冒险游戏，背景设定在克劳狄斯的皇宫中。玩家将扮演六个角色之一，在三天内完成各自的目标。

## 游戏角色

游戏中有六个可选择的角色，每个角色都有独特的身份、初始装备和目标。

1. **克劳狄斯** - 国王：威严的国王，掌握着王国的最高权力
2. **程婴** - 御医：医术高超但风评阴狠的御医，传说就是他陷害死了公孙杵臼
3. **何非** - 潜水教练：潜水教练，消息非常灵通，什么人都认识。写作江湖百晓生读作江湖骗子
4. **李想** - 市政给排水工程师：代表作是建设过新城
5. **吴智哲** - 建筑设计师：曾经的王宫设计大师的关门弟子
6. **王卫国** - 施工运输队队长：运输队班长，熟悉王宫里的线路。

## 玩法说明

- **角色选择**：游戏开始时，从六个角色中选择一个开始游戏
- **三天时限**：游戏共进行三天，每天是一或多个章节
- **装备交换**：触发时可以与其他角色交换一次装备，通过策略性的交换来获得完成目标所需的物品
- **目标达成**：在三天内收集到完成角色目标所需的装备即为成功，否则失败
- **策略性**：需要仔细规划每天的策略，每一个选择都有可能影响结局
